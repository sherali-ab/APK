## 📱 Экраны приложения

### 🔐 Авторизация и регистрация

---

#### `SplashScreen` — Экран загрузки
Первый экран при открытии приложения. Проверяет наличие сохранённой сессии:
- Если токен есть → загружает профиль пользователя и переходит к заданиям
- Если токена нет → переходит на экран входа

---

#### `LoginScreen` — Вход в аккаунт
Экран авторизации пользователя.

**Функции:**
- Вход по логину/email + пароль
- Показ/скрытие пароля
- Переход к регистрации
- Переход к восстановлению пароля
- Сохранение сессии через `AccountService`
- Автоматическая отправка FCM-токена после входа

---

#### `RegistrationScreen` — Регистрация
Создание нового аккаунта.

**Функции:**
- Поля: имя, username, email, пароль, подтверждение пароля
- Проверка доступности username в реальном времени (debounce 500мс)
- Валидация формы
- Отправка кода подтверждения на email
- Переход к верификации email

---

#### `VerifyEmailScreen` — Подтверждение email
Ввод 6-значного кода из письма.

**Функции:**
- Ввод OTP кода
- Повторная отправка кода
- Подтверждение аккаунта и переход к выбору роли

---

#### `ForgotPasswordScreen` — Восстановление пароля
Сброс пароля через email.

**Функции:**
- Ввод email
- Отправка кода сброса
- Установка нового пароля

---

### 👤 Настройка профиля

---

#### `RoleRegistrationScreen` — Выбор роли
Пошаговая настройка профиля после регистрации. Пользователь выбирает роль и заполняет данные.

**Роли:**
- 👔 **Работодатель** — размещает задания
- 🔧 **Исполнитель** — откликается на задания

**Шаги настройки:**

| Шаг | Экран | Описание |
|-----|-------|----------|
| 1 | `NameStepScreen` | Ввод полного имени и телефона |
| 2 | `ContactsStepScreen` | Контактные данные |
| 3 | `CategoriesStepScreen` | Выбор категорий специализации |
| 4 | `PhotoStepScreen` | Загрузка фото профиля |
| 5 | `BioStepScreen` | Краткое описание о себе |

---

### 🏠 Главный экран

---

#### `MainMenuScreen` — Главное меню
Стартовый экран после авторизации. Отображает профиль пользователя и основную навигацию.

**Функции:**
- Отображение аватара, имени, роли
- Быстрый доступ ко всем разделам
- Значок Premium-статуса
- Переключение между аккаунтами

---

### 📋 Задания

---

#### `TasksScreen` — Лента заданий
Основной экран с лентой доступных заданий.

**Функции:**
- Список всех активных заданий из БД
- Фильтрация по категориям (Ремонт, IT, Красота и др.)
- Поиск по заданиям
- Анимированный градиентный фон
- Для работодателей — кнопка создания задания
- Для исполнителей — просмотр и отклик
- Навигация: Профиль, Чаты, Уведомления, AI-помощник, Аналитика

---

#### `CreateTaskScreen` — Создание задания
Форма создания нового задания (только для работодателей).

**Поля формы:**
- Название и описание задания
- Категория (из 10+ категорий)
- Тип оплаты: фиксированная / почасовая
- Бюджет (₽)
- Место выполнения: у заказчика / у исполнителя / удалённо
- Дедлайн (дата и время)
- Адрес
- Контактный телефон и email
- Срочность задания
- Прикрепление фото (до нескольких изображений)

---

#### `TaskDetailsScreen` — Детали задания
Подробная информация о задании.

**Функции:**
- Полное описание задания
- Профиль заказчика с рейтингом и отзывами
- Кнопка «Откликнуться» → открывает чат с заказчиком
- Просмотр фотографий задания
- Метки: срочность, тип оплаты, место выполнения
- Для заказчика: просмотр откликов исполнителей

---

#### `MyProjectsScreen` — Мои проекты
Список заданий пользователя (созданных или взятых в работу).

**Функции:**
- Задания по статусам: активные, в работе, завершённые
- Управление своими заданиями (для работодателя)
- История выполненных заданий (для исполнителя)

---

### 💬 Чаты

---

#### `ChatsListScreen` — Список чатов
Все диалоги пользователя.

**Функции:**
- Список чатов с превью последнего сообщения
- Счётчики непрочитанных сообщений
- Режим редактирования (удаление чатов)
- Архивные чаты
- Premium-значок у собеседников

---

#### `ChatScreen` — Экран чата
Полнофункциональный мессенджер внутри приложения.

**Функции:**
- Текстовые сообщения в реальном времени
- 🎤 Голосовые сообщения (запись и воспроизведение)
- 📹 Видео-кружочки (запись и отправка)
- 📎 Прикрепление файлов
- 🖼 Отправка изображений с превью
- Просмотр профиля собеседника
- Push-уведомления через Firebase FCM
- Индикаторы прочтения

---

#### `ChatInfoScreen` — Информация о чате
Детальная информация о диалоге и собеседнике.

---

### 👤 Профиль

---

#### `ProfileScreen` — Мой профиль
Личная страница пользователя.

**Функции:**
- Фото, имя, username, роль
- Рейтинг и отзывы
- Статистика (выполненные/созданные задания)
- Premium-статус с датой истечения
- Переключение между несколькими аккаунтами
- Переход к настройкам

---

#### `UserProfileScreen` — Профиль другого пользователя
Публичная страница любого пользователя.

**Функции:**
- Фото, имя, bio, категории
- Рейтинг и список отзывов
- Кнопка «Написать» → открывает чат
- История выполненных работ

---

#### `ReviewsScreen` — Отзывы
Список отзывов пользователя с оценками.

---

#### `SettingsScreen` — Настройки
Редактирование профиля и параметров аккаунта.

**Функции:**
- Изменение имени, username, телефона, email
- Загрузка/смена фото профиля
- Смена пароля с подтверждением по email (OTP)
- Проверка доступности username в реальном времени
- Редактирование описания «О себе»
- Выбор и обновление категорий

---

### ⭐ Premium

---

#### `PremiumScreen` — Подписка Premium
Экран оформления Premium-подписки.

**Тарифы:**

| Тариф | Срок | Стоимость |
|-------|------|-----------|
| Месячный | 30 дней | 10 ₽ |
| Квартальный | 90 дней | 10 ₽ |
| Годовой | 365 дней | 10 ₽ |

**Функции:**
- Выбор тарифного плана
- Оплата через ЮMoney (QR-код / СБП)
- Просмотр договора оферты (PDF)
- Проверка статуса платежа каждые 3 секунды
- Автоматическая активация Premium после оплаты (webhook)
- Отображение даты истечения подписки

**Преимущества Premium:**
- Значок Premium в профиле и чатах
- Расширенная аналитика
- Приоритет в ленте заданий

**Функции:**

- Помощь в поиске заданий

## 🔌 API

### Основные эндпоинты

| Метод | Путь | Описание | Auth |
|-------|------|----------|------|
| POST | `/auth/register` | Регистрация | ❌ |
| POST | `/auth/login` | Вход | ❌ |
| POST | `/auth/verify-email` | Подтверждение email | ❌ |
| POST | `/auth/forgot-password` | Сброс пароля | ❌ |
| GET | `/profile/:id` | Получить профиль | ✅ |
| PUT | `/profile/update` | Обновить профиль | ✅ |
| GET | `/tasks` | Список заданий | ✅ |
| POST | `/tasks` | Создать задание | ✅ |
| GET | `/tasks/:id` | Детали задания | ✅ |
| GET | `/chats` | Список чатов | ✅ |
| GET | `/chats/:id/messages` | Сообщения чата | ✅ |
| POST | `/chats/:id/messages` | Отправить сообщение | ✅ |
| GET | `/notifications` | Уведомления | ✅ |
| POST | `/premium/create-qr` | Создать QR для оплаты | ✅ |
| GET | `/premium/payment-status/:orderId` | Статус платежа | ✅ |
| POST | `/premium/webhook` | Webhook ЮMoney | ❌ |
| GET | `/analytics` | Аналитика | ✅ |
| POST | `/ai/support` | AI-помощник | ✅ |

## 🔒 Безопасность

- JWT токены для аутентификации всех защищённых запросов
- Bcrypt для хеширования паролей
- SHA1 подпись для верификации webhook-уведомлений ЮMoney
- HTTPS через Let's Encrypt (обязательно для webhook)
- Middleware проверки токена на всех защищённых роутах
